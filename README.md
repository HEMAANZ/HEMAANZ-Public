# HEMA Aotearoa New Zealand (HEMAANZ)

Living documentation and policies published by HEMAANZ.

Draft policies are made available for public viewing and consultation, during this phase the draft can be reformed according to consult advice.

Draft policies that successfully pass the Consultation phase are finally decided upon by a General Meeting or Committee vote, depending on whether they place any obligations on members. 
Upon approval, they are promoted to Published policies. 
Relevant policies must be adhered to for clubs to become mMembers and events to be endorsed.

Documentation is open to submissions from HEMAANZ members via pull request or email.
Submissions and other edits are able to be applied by executive members - all changes should follow the processes defined in the constitution and consultation policy.

Documents will be written and published in markdown: [https://www.markdownguide.org](https://www.markdownguide.org)

## Why are we using Github?

A software development platform may not appear the most intuitive platform for hosting documentation for a union of sword clubs, but Git (and it's most popular online host Github) offers a number of benefits for hosting live documents:

1. Free hosting. 
2. Open plaintext-based formatting that is easy to edit. 
3. Tracked changes and version control, so everything's kept transparent. 
4. Configurable change proposal and approval processes baked into the platform if desired. 
5. Capacity to incorporate a CI pipeline that will publish these documents to an actual website automatically.
